O html só para exibição dos thumbnails

Caso você queira apenas exibir os thumbnails sem a necessidade de exibir algum conteúdo ao ser clicado, deve-se setar isso na configuração do plugin. Ex: 

							$(document).ready(function(){
								$('.my-grid').WMGridfolio({
									config : {
										onlyThumb : true
									}
								});
							});
